Технічне завдання

Створення REST API з кінцевими точками CRUD (створення, читання, оновлення, видалення) для "команд" і "людей в цих командах".

Стек

  • WEB-сервер: Django
  • База даних: PostgreSQL 13
  • Контейнеризація: Docker
  • OPEN API Документація: Yasg/Swagger
